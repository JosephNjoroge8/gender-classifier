🆘 TROUBLESHOOTING:
═══════════════════════════════════════════════════════════════════════════════

If you get 500 error after upload:
  → Check /storage/logs/laravel.log for details
  → Verify file permissions (should be 755)
  → Ensure database.sqlite file uploaded correctly

If you get 404 error:
  → Verify public/index.php uploaded
  → Verify routes/api.php uploaded
  → Check that files are in /public_html not /public_html/public_html

If you get timeout:
  → Genderize.io API might be slow
  → API has 15-second timeout built in
  → Check server logs for details
